我的第一篇博客-博客创建之路
2024/10/7

第一章 - 选择博客平台

可以选择使用现有的博客平台，或者自己搭建一个独立的博客。这里有几种常见的选择：

使用博客平台（简单、快速）
WordPress.com：不需要太多技术知识，可以快速创建和发布博客。平台提供托管服务，有免费和付费套餐。
Medium：适合想专注于写作而不关心技术搭建的人，提供简洁的界面，容易上手。
Blogger：Google 提供的免费博客平台，操作简单，适合初学者。
自托管博客（更多自定义、专业）
如果你想完全掌控自己的博客网站并能够进行更多的自定义，可以选择自托管方式。

WordPress.org：你需要购买域名和主机，并安装 WordPress。WordPress 是一个强大的内容管理系统（CMS），支持自定义主题和插件，灵活性很高。
Jekyll + GitHub Pages：Jekyll 是一个静态网站生成器，适合有一定技术背景的开发者。可以通过 GitHub Pages 免费托管，非常适合开发者展示技术博客。
Hexo：Hexo 是一个基于 Node.js 的静态博客生成器，速度快、简单易用，适合前端开发者使用。

第二章 - 选择域名和主机

如果你选择自托管博客，以下步骤至关重要：

购买域名：选择一个代表你品牌或个人风格的域名。可以使用域名注册商如 GoDaddy、Namecheap 等购买域名。
选择主机服务：购买适合你博客流量和内容需求的主机服务，如：
Bluehost：适合新手用户，简单易用。
SiteGround：WordPress 推荐的主机之一，速度和安全性较好。
GitHub Pages：适合使用 Jekyll 或 Hexo 搭建静态博客，提供免费的托管服务。

2. 为什么是 line，而不是 bash？

代码块的语言标记（如 bash、python、javascript 等）应该在代码块的开始部分定义。如果在你的主题或 Markdown 渲染设置中，将语言标记设置为 line，则可能是以下几个原因：

  • 主题限制：某些主题可能不支持所有语言的高亮，可能会默认显示为 line。你可以检查你所使用的主题的文档，了解它对代码高亮的支持情况。

  • 配置设置：检查 Hexo 配置文件（通常是 _config.yml），看看是否有关于代码高亮的设置。如果你使用的是较新的 Hexo 版本（7.0.0 及以上），可以直接配置为：

    $ syntax_highlighter: prismjs

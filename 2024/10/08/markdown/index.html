Markdown 简介

2024/10/8

Markdown 是一种轻量级的标记语言，设计初衷是以简单、易读易写的方式编写结构化的文档内容。它的语法规则非常直观，甚至在不转换的情况下也可以清晰地阅读，适合写作博客、文档、README 文件等。Markdown 编写的文件通常使用 .md 或 .markdown 后缀。

Markdown 的应用场景
技术文档：Markdown 被广泛应用于编写技术文档、项目说明等。
博客：如 Hexo、Jekyll 等静态网站生成器都支持 Markdown 语法。
协作平台：GitHub、GitLab 等平台的 README 文件通常使用 Markdown 编写，便于开发者之间的协作和交流。
Markdown 简单直观，非常适合需要快速编写和组织内容的场景。

常用语法示例

  1. 标题：使用 # 表示标题，# 的数量表示标题的级别。
# 一级标题
## 二级标题
### 三级标题
  1. 列表：
    无序列表：使用 -、* 或 +。
- 项目1
- 项目2
- 项目3

有序列表：使用数字加 .。

1. 第一项
2. 第二项
3. 第三项

  1. 粗体和斜体：
    粗体：**粗体** 或 __粗体__
    斜体：*斜体* 或 _斜体_
    粗斜体：***粗斜体***

  2. 链接：

[链接文本](https://example.com)
  1. 图片：
![图片描述](图片链接)
  1. 引用：使用 > 表示引用。
> 这是一个引用

7.
行内代码：使用反引号 ` 包裹。

这是 `inline code` 示例。

多行代码块：使用三个反引号 或缩进4个空格。

```inline code```

    inline code

| 头部1 | 头部2 |
